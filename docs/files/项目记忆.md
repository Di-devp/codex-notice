# 项目记忆

## 2026-06-11 Notice 当前关键架构记忆

### 技术栈
- 桌面应用：Tauri 2 + Rust + Tokio。
- 前端：Vue 3 + TypeScript + Vite + Pinia + Naive UI。
- 本地 HTTP：Axum，监听 `127.0.0.1:3746`。
- 存储：SQLite，默认路径为 `~/Library/Application Support/Notice/notice.db`。
- 敏感配置：飞书 webhook 与签名密钥走 macOS Keychain；SQLite 中只保存非敏感配置和脱敏展示信息。

### Hook 事件约定
Notice 管理的 Codex hooks 必须包含：

- `SessionStart`
- `UserPromptSubmit`
- `PermissionRequest`
- `PreToolUse`
- `PostToolUse`
- `Stop`

`UserPromptSubmit` 是红绿灯进入“运行中”的关键事件。缺少该事件时，用户新发消息不会被识别为新一轮任务开始。

### 红绿灯状态优先级
状态计算入口：`src-tauri/src/storage/mod.rs` 的 `traffic_widget_status()`。

优先级如下：

1. 红灯：最近 30 分钟内任一会话最新有效状态为失败。
2. 黄灯：无失败时，存在 pending approval 或用户确认状态。
3. 跑马灯：无失败、无待审批时，存在未结束的活动会话。
4. 绿灯：没有失败、没有待审批、没有运行中会话。

失败会话收到新的 `TASK_START` 后，会重新进入运行态，红灯消失。

### 飞书通知策略
飞书只通知两类事件：

- 需要用户审批。
- 一个会话中的任务成功结束。

工具调用过程中的普通成功事件不通知，失败或未完成的 Stop 不发送成功通知。

### 常用验证命令
- 前端构建：`pnpm build`
- Rust 测试：`cargo test --manifest-path src-tauri/Cargo.toml`
- Debug 打包：`pnpm tauri build --debug`
- 健康检查：`curl -sS http://127.0.0.1:3746/health`

### 用户当前需要注意
如果 `~/.codex/config.toml` 中搜索不到 `UserPromptSubmit`，需要在 Notice 的 Providers 页面重新安装 Codex Hooks，并新开 Codex 会话或重启 Codex 客户端测试。

## 2026-06-16 Codex 用量展示架构记忆

### 数据来源
- Codex 当前剩余用量没有接入公开个人额度 API。
- Notice 通过本机 Codex 会话日志读取最近一次 `rate_limits`：
  - `~/.codex/sessions/**/*.jsonl`
  - `~/.codex/archived_sessions/**/*.jsonl`
- 解析入口：`src-tauri/src/codex_usage.rs`。
- 只解析包含 `"rate_limits"` 的 JSONL 行，不读取或展示 auth token。

### 状态对象
`TrafficWidgetStatus` 现在除原红绿灯字段外，还包含：

- `codex_usage.limit_id`
- `codex_usage.limit_name`
- `codex_usage.primary`
- `codex_usage.secondary`
- `codex_usage.plan_type`
- `codex_usage.rate_limit_reached_type`
- `codex_usage.updated_at`

前端 camelCase 字段为 `codexUsage`，定义在 `src/types.ts`。

### 展示约定
- `src/App.vue` 的 `codexUsageText` 将 primary/secondary 窗口格式化为 `5h 100% · 7d 100%`。
- 小组件状态优先级仍由红灯、黄灯、跑马灯、绿灯决定；用量只是附加信息，不改变状态优先级。
- 小组件最终窗口尺寸为 `232x76`；根容器、组件 CSS、Tauri 窗口约束需要保持一致。

### 常用打包命令
- Release 打包：`pnpm tauri:build`
- 产物路径：
  - `src-tauri/target/release/bundle/macos/Notice.app`
  - `src-tauri/target/release/bundle/dmg/Notice_1.0.0_aarch64.dmg`
